# Assignment 1 - Personal Webpage

This site is a public personal webpage built for Assignment 1. It reuses content from Lab 1 and adds expanded sections, improved styling, and a responsive layout for desktop and mobile.

## Contents
- `index.html` - main page
- `styles.css` - styling
- `script.js` - small enhancements for scroll reveal and year

## Local preview
1. Open `index.html` in a browser.

## Deploy to GitHub Pages
1. Create a public repository named `yourusername.github.io` on GitHub.
2. Open PowerShell in this folder.
3. Run:
   git init
   git add .
   git commit -m "Initial commit for personal webpage"
   git branch -M main
   git remote add origin https://github.com/yourusername/yourusername.github.io.git
   git push -u origin main

4. Your site will be live at `https://yourusername.github.io/`.

## Submission
- GitHub Pages URL
- GitHub repository link
